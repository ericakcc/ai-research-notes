# World Models

Learning compressed spatial-temporal models of environments, then training agents entirely within these learned "dreams."

## Learning Path

1. Implement the VMC pipeline — VAE for perception, MDN-RNN for dynamics, Controller for action
2. Understand latent space as information bottleneck
3. Progress to DreamerV3 and diffusion-based world models
